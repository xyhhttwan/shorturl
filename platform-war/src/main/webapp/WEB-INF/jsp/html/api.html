欢迎使用API接口，生成稳定、可靠的短网址!

TXT格式短网址API接口

接口：http://mrw.so/api.php?url=urlencode('要缩短的网址')

例如：http://mrw.so/api.php?url=http%3A%2F%2Fwww.baidu.com

返回：http://mrw.so/baidu

JSON格式短网址API接口

说明：format为json

例如：http://mrw.so/api.php?format=json&url=http%3A%2F%2Fwww.baidu.com

返回：{"url":"http://mrw.so/baidu","err":""}

JSONP格式短网址API接口

说明：format为jsonp

例如：http://mrw.so/api.php?format=jsonp&url=http%3A%2F%2Fwww.baidu.com&callback=callbackname

返回：callbackname({"url":"http://mrw.so/baidu","err":""})
